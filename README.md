## node notice server by socketio   
> Requirements  
1. node 6.0+  
2. socketio  
3. koa2   
4. mysql  
5. redis  
6. bunyan

### Install    
```  
   npm install
```

###  Run      
> see the config file: config_dev.json  

* config mysql && start    
* config redis && start 
* run
```  
   npm start  
```
